# 世界观构建清单

本文档提供世界观设计的完整框架。不是所有小说都需要填满每一项，根据题材选择适用的部分。

---

## 1. 时空背景

### 时间设定
- [ ] 纪年方式（公历/架空纪元/帝王年号）
- [ ] 故事发生在什么时代（远古/上古/中古/近古/近代/现代/未来）
- [ ] 科技水平（石器/青铜/铁器/火药/蒸汽/电气/信息/星际）
- [ ] 时间跨度（故事跨越多久：几天/几年/几代人）

### 空间设定
- [ ] 世界形态（单一大陆/群岛/星球/多重位面/虚拟空间）
- [ ] 地理格局（大陆板块、海洋、山脉、河流的分布）
- [ ] 关键地标（故事涉及的重要地点）
- [ ] 气候带分布（影响文明分布和战争形态）

### 文明形态
- [ ] 主导文明类型（农耕/游牧/海洋/商业/修仙/魔法/科技）
- [ ] 文明程度（蒙昧/发展中/成熟/衰落/复兴）
- [ ] 多文明关系（孤立/交流/冲突/融合）

---

## 2. 社会体系

### 政治体制
- [ ] 统治形式（帝制/分封/联邦/共和/宗门联盟/城邦/无政府）
- [ ] 权力结构（中央集权/地方分权/教权/军权/神权）
- [ ] 权力传承方式（世袭/禅让/选举/实力决定/血统决定）
- [ ] 政治派系（主要的权力斗争方）

### 社会阶层
- [ ] 阶层划分（贵族/平民/奴隶/修行者等级……）
- [ ] 阶层流动性（能否跨越阶层，通过什么途径）
- [ ] 特殊群体（修行者/异族/奴隶/流民/商人的社会地位）
- [ ] 阶层矛盾（主要的社会冲突）

### 经济体系
- [ ] 货币体系（货币种类、汇率、购买力）
- [ ] 主要产业（农业/手工业/商业/矿业/修行资源）
- [ ] 贸易路线（商路、贸易城市、贸易壁垒）
- [ ] 资源分布（关键资源的地理分布，可能引发的争夺）

### 文化习俗
- [ ] 信仰体系（宗教/图腾/祖先崇拜/无神论）
- [ ] 礼仪规范（社交礼仪、等级礼仪、婚丧嫁娶）
- [ ] 禁忌（文化禁忌、宗教禁忌、政治禁忌）
- [ ] 节日庆典（重要节日及其含义）
- [ ] 教育体系（知识传承方式、学校/师徒/自学）
- [ ] 语言文字（通用语、方言、古语、密语）

---

## 3. 力量/规则体系

### 修炼/能力体系
- [ ] 力量来源（灵气/魔力/科技/血脉/信仰/自然之力）
- [ ] 等级划分（每个等级的名称、特征、标志性能力）
- [ ] 修炼方式（打坐/战斗/悟道/炼丹/科技改造/血统觉醒）
- [ ] 突破条件（升级需要什么：资源/机缘/战斗/顿悟）
- [ ] 天赋差异（天赋如何影响上限，是否公平）
- [ ] 战力天花板（最强者有多强，是否存在限制）

### 特殊规则
- [ ] 世界底层规则（因果律/轮回/天道/命运是否存在）
- [ ] 禁忌领域（不可触碰的力量/知识/区域）
- [ ] 失衡机制（力量过大是否招致反噬/天劫/制裁）

### 平衡设计
- [ ] 力量克制关系（是否有相生相克）
- [ ] 弱者生存之道（低级者如何在高级者的世界中生存）
- [ ] 力量的代价（获得力量需要付出什么）

---

## 4. 势力阵营

### 主要势力（每个势力填写以下信息）

| 字段 | 说明 |
|------|------|
| 名称 | 势力名称 |
| 领袖 | 核心领导者 |
| 理念/宗旨 | 势力的核心信念 |
| 实力等级 | 在世界中的实力排名 |
| 地盘 | 控制的区域 |
| 核心优势 | 最强的方面（兵力/财富/技术/情报/……） |
| 内部矛盾 | 势力内部的分歧或隐患 |
| 与主角的关系 | 初期关系 → 可能的变化方向 |

### 势力关系矩阵

绘制势力间的关系：
- 盟友 / 敌对 / 中立 / 暗中合作 / 从属 / 暗中敌对
- 关系的历史原因
- 关系可能的变化节点（与故事主线的关联）

---

## 5. 历史脉络

### 重大历史事件
按时间线列出影响当前格局的重大事件：
- 远古传说（可能影响世界观底层设定）
- 中古变局（塑造当前势力格局的事件）
- 近代冲突（直接导致当前矛盾的事件）
- 故事开端前夕的导火索

### 传说与秘辛
- 未被证实的传说（可能是伏笔）
- 被掩盖的历史真相
- 失落的文明/力量/知识

### 历史遗留问题
- 当前冲突的历史根源
- 未解决的历史恩怨
- 被遗忘但可能重演的历史模式

- [ ] 普通人的一天是什么样的
- [ ] 食物和饮食习惯
- [ ] 交通工具和出行方式
- [ ] 通讯方式（信鸽/魔法通讯/网络/口信）
- [ ] 医疗水平
- [ ] 娱乐方式
- [ ] 建筑风格

## 自洽性检查

完成世界观设计后，检查以下问题：

- [ ] 力量体系是否与社会结构匹配？（如：个体强者能毁城灭国，社会还会是帝制吗？）
- [ ] 经济体系是否支撑得起所述的战争/修炼消耗？
- [ ] 势力格局是否平衡？（是否存在一个碾压性的势力让故事无法展开）
- [ ] 历史事件是否解释了当前的矛盾和格局？
- [ ] 规则体系是否有内在逻辑？（不能想一出是一出）
- [ ] 是否为后续情节留出了足够的空间？（不要把世界设定得太满太死）
- [ ] 普通人的生活在力量体系存在的情况下是什么样的？（避免只有主角团的世界）
