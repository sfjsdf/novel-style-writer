# 日常纲指南

日常纲是**过渡章**的标准化写法，避免日常章变成"水章"。

---

## 什么是过渡章

- 两个高潮之间的缓冲期
- 修炼/成长/积累的过程
- 人物关系的日常互动
- 信息收集/推理的过程
- 新环境的适应期

---

## 日常章的核心原则

### 不能"水"

每章必须至少做到以下之一：
1. **推进一个信息**：揭露一个新信息、回收一个伏笔
2. **推进一个人物关系**：关系发生变化（哪怕很微小）
3. **推进一个设定**：展示世界观的新层面
4. **推进一个悬念**：埋下新钩子或推进旧悬念

### 必须有钩子

即使是日常章，结尾也必须有钩子。强度可以是B级或C级，但不能没有。

---

## 日常章的固定结构

### 结构模板

```
日常章 = 互动场景 + 信息/铺垫 + 钩子
```

**互动场景**（60-70%篇幅）：
- 人物之间的对话和互动
- 展现人物性格和关系
- 推进日常情节

**信息/铺垫**（20-30%篇幅）：
- 透露一个新信息
- 或为后续事件做铺垫
- 或展示世界观的新层面

**钩子**（10%篇幅）：
- 章末留一个悬念
- 可以是B级（好奇）或C级（轻微）

### 日常章的三种变体

#### 变体A：人物塑造型

重点：通过日常互动展现人物性格

```
场景：师徒在药铺的日常
- 阿青的小习惯（舔嘴唇、抠指甲）
- 沈渡的克制（喝茶、碾药）
- 一个有趣的互动（阿青问问题，沈渡回避）
→ 信息：阿青提到自己做了个梦
→ 钩子：沈渡听到后沉默了很久
```

#### 变体B：世界观展示型

重点：通过日常场景展示世界观

```
场景：沈渡去镇上赶集
- 镇上的物价（展示经济体系）
- 遇到江湖人（展示势力分布）
- 听到传闻（展示历史背景）
→ 信息：传闻中提到一个熟悉的名字
→ 钩子：沈渡的脸色变了
```

#### 变体C：伏笔推进型

重点：推进一个悬而未决的线索

```
场景：阿青独自调查
- 去茶馆打听消息
- 发现一个细节
- 回来告诉沈渡
→ 信息：死者与沈渡有某种关联
→ 钩子：沈渡决定今晚出去
```

---

## 日常章的检查清单

- [ ] 是否至少推进了一个信息/关系/设定/悬念？
- [ ] 是否有章末钩子？
- [ ] 人物互动是否展现了性格？
- [ ] 是否有"水"的感觉？（连续多段无新信息）
- [ ] 是否为后续事件做了铺垫？
- [ ] 对话是否有信息量或推动情节？
- [ ] 是否适配手机阅读（段落不超过4-5行）？

---

## 避免"水章"的技巧

1. **每段对话必须有信息量**：废话删掉
2. **环境描写不超过2-3句**：服务于氛围，不炫技
3. **心理描写用动作代替**：不写"他想"，写"他做了什么"
4. **每个场景必须有目的**：删掉"为写而写"的场景
5. **日常中藏暗流**：表面平静，底下有信息在流动
