# 整体审校清单

全书完成后，使用本清单进行系统性审校。逐项检查，记录问题和修改方案。

---

## 0. 网络小说专项检查

### 黄金三章
- [ ] 第一章是否在500字内抓住读者？
- [ ] 前三章是否建立了主角、核心冲突和世界设定？
- [ ] 前三章是否有至少一个爽点？

### 爽点分布
- [ ] 是否每章都有爽点或情绪高点？
- [ ] 是否有连续3章以上无爽点？
- [ ] 爽点类型是否多样化（不重复同一种爽感）？
- [ ] 爽点强度是否有递进（不能一直同一个强度）？

### 章末钩子
- [ ] 每章结尾是否有钩子？
- [ ] 钩子类型是否多样（悬念/反转/危机/揭秘/情感）？
- [ ] 钩子是否足够有力（读者会不会想翻下一章）？

### 可读性
- [ ] 段落是否适合手机阅读（单段不超过4-5行）？
- [ ] 对话占比是否在30-40%？
- [ ] 是否有连续超过500字无事件推进的段落？
- [ ] 节奏是否偏快（少铺垫多推进）？

### 人物
- [ ] 主角是否有鲜明的记忆点（性格标签+独特能力/习惯）？
- [ ] 主角的"金手指"是否明确？
- [ ] 反派是否有魅力和逻辑？

---

## 1. 连贯性检查

### 时间线
- [ ] 故事内时间是否连续？有无矛盾（如"三天后"但事件量超出三天）？
- [ ] 各章的时间标注是否一致？
- [ ] 回忆/闪回的时间标注是否清晰？
- [ ] 不同视角线的时间是否对齐（交叉叙事时特别注意）？

### 空间线
- [ ] 人物移动距离与所用时间是否匹配？
- [ ] 地点描写前后是否一致（同一地点在不同章节的描写有无矛盾）？
- [ ] 地理关系是否合理（A 在 B 北边，B 在 C 东边，不能写成 A 在 C 南边）？

### 物品/道具
- [ ] 重要道具的获取、使用、丢失是否有交代？
- [ ] 是否有"凭空出现"的物品？
- [ ] 武器/法宝的能力描述是否前后一致？

---

## 2. 人物一致性

### 言行一致
- [ ] 人物在不同场景中的性格表现是否一致？
- [ ] 人物的知识范围是否合理（不该知道的是否知道了）？
- [ ] 人物的能力水平是否与设定匹配（不会突然变强/变弱无解释）？
- [ ] 人物的语言风格是否稳定（同一角色不能一会儿文雅一会儿粗俗，除非有原因）？

### 人物弧光
- [ ] 人物的成长变化是否有足够的铺垫？
- [ ] 变化是否过于突兀（缺乏过渡）？
- [ ] 每个主要角色的弧光是否完整闭合？
- [ ] 配角是否在完成故事功能后有合理的退场？

### 角色区分度
- [ ] 不同角色的说话方式是否有区分度？
- [ ] 去掉角色名，读者能否通过对话判断是谁在说话？
- [ ] 是否有"千人一面"的问题（多个角色读起来像同一个人）？

---

## 3. 伏笔与呼应

### 伏笔回收
- [ ] 列出所有已埋伏笔，逐一检查是否已回收
- [ ] 回收的时机是否合适（不能太早也不能遗忘）？
- [ ] 回收方式是否自然（不能突然跳出来说"还记得之前那个吗"）？
- [ ] 是否有"哑弹"（埋了伏笔但最终没有用上）？

### 伏笔质量
- [ ] 伏笔是否足够隐蔽（读者第一遍不会注意到）？
- [ ] 回收时是否有"原来如此"的惊喜感？
- [ ] 伏笔与回收之间的间隔是否合适（不能太近无悬念，太远被遗忘）？

### 前后呼应
- [ ] 开篇与结尾是否有呼应？
- [ ] 重要场景/台词是否有回响（在后文以变体形式重现）？
- [ ] 主题是否有贯穿全书的意象/母题？

---

## 4. 风格一致性

### 与风格画像对比
- [ ] 全书的叙事视角是否稳定（无意外的人称切换）？
- [ ] 句式特征是否与风格画像一致？
- [ ] 用词水平是否稳定（无突然的古白切换）？
- [ ] 对话风格是否一致？
- [ ] 节奏变化是否在风格画像的框架内？

### 前后统一
- [ ] 第一章与最后一章的风格是否有明显偏差？
- [ ] 中间章节是否有风格漂移（写着写着跑偏）？
- [ ] 不同卷之间风格是否统一（除非有意设计的风格演变）？

---

## 5. 情节逻辑

### 因果链
- [ ] 每个重大事件是否有合理的前因？
- [ ] 人物的行为动机是否充分？
- [ ] 是否有"剧情需要"式的不合理巧合？
- [ ] 反派的计划是否有逻辑（不能太蠢也不能全知全能）？

### 节奏与张力
- [ ] 全书的张力曲线是否合理（是否有长时间的平淡段）？
- [ ] 高潮是否足够有力（是否是全书最有张力的部分）？
- [ ] 低谷段是否有足够的铺垫价值（不是无意义的拖沓）？
- [ ] 各卷的高潮是否在递进（不能一卷比一卷弱）？

### 副线处理
- [ ] 所有副线是否都有交代？
- [ ] 副线是否喧宾夺主（占用过多篇幅）？
- [ ] 副线与主线的交汇是否自然？

---

## 6. 结构完整性

### 开篇
- [ ] 第一章是否能抓住读者？
- [ ] 核心冲突是否在前几章有所暗示？
- [ ] 主角是否在前几章就让读者产生兴趣/共情？

### 结尾
- [ ] 结局是否令人满意（不一定是大团圆，但要有交代）？
- [ ] 是否有"烂尾"感（仓促/敷衍/未解之谜太多）？
- [ ] 结尾的节奏是否合适（不能太赶也不能拖沓）？
- [ ] 最后一段/一句是否有力量？

### 整体
- [ ] 全书字数是否在预期范围内？
- [ ] 各卷字数是否均衡（不能一卷特别长一卷特别短，除非有意设计）？
- [ ] 章节标题是否有风格感？

---

## 7. 最终确认

- [ ] 以上所有检查项的问题是否已修复？
- [ ] 全文是否有错别字/标点错误？
- [ ] 人名/地名是否前后一致（不能同一角色两个名字，除非有意设计）？
- [ ] 是否满意可以定稿？
