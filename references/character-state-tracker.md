# 人物状态追踪表

每章写作完成后更新此表，确保人物状态前后一致。

---

## 使用方法

每章结束后，检查本章中出场的角色，更新其状态。如果某角色本章未出场，状态不变，无需更新。

## 更新规则

1. **每章结束后必须更新**出场角色的状态
2. **只更新有变化的字段**，没变化的保持原样
3. **"知道什么"和"不知道什么"最重要**——信息差是悬疑小说的核心，一旦搞错就会出现逻辑漏洞
4. **武功状态**：如果角色受伤/突破/退步，必须记录原因和章节
5. **物品追踪**：重要道具的获取、使用、转手必须记录

---

## 新角色添加

当新角色出场时，在本表中新增一个条目，填写初始状态。

---

## 示例更新格式

```
## 第X章结束后更新

### 沈渡
- 所在位置：渡口镇·药铺 → 镇东屠场（第X章去了现场）
- 知道什么：命案手法是摘星手 → 确认凶手在模仿自己的手法（第X章二次勘查）
- 情绪状态：压抑 → 震惊+压抑（第X章确认后）

### 阿青
- 知道什么：师父不是普通人 → 察觉师父对命案有异常反应（第X章观察到）
```
