# 伏笔追踪表

记录全书所有伏笔的埋设和回收状态，确保没有遗漏。

---

## 使用方法

1. **埋设伏笔时**：在本表新增条目，记录内容、埋设章节、计划回收章节
2. **回收伏笔时**：更新状态为"已回收"，记录实际回收章节
3. **每章写作前**：检查本章是否有需要回收的伏笔

## 状态说明

- **待回收**：已埋设，等待回收
- **已回收**：已回收，记录实际回收章节
- **进行中**：贯穿全书的意象/母题
- **废弃**：因情节调整不再需要的伏笔（需说明原因）

---

## 检查清单

### 每章写作前检查
- [ ] 本章计划回收哪些伏笔？（查看"计划回收章节"列）
- [ ] 这些伏笔在前面章节是否已经埋好？

### 每卷结束时检查
- [ ] 本卷计划回收的伏笔是否全部回收？
- [ ] 是否有伏笔被遗忘（超过计划回收章节仍未回收）？
- [ ] 是否有新伏笔需要埋设？

### 全书结束时检查
- [ ] 所有"待回收"伏笔是否已变为"已回收"？
- [ ] 是否有哑弹（埋了但没用上的伏笔）？
- [ ] 贯穿全书的意象/母题是否完整闭合？

## 伏笔分类

### 剧情伏笔
指向具体事件/真相的伏笔，必须精确回收。

示例：
- "沈渡认出摘星手" → 后续揭示他就是这个手法的使用者
- "阿青没有记忆" → 后续揭示其身世

### 人物伏笔
指向角色身份/关系/秘密的伏笔。

示例：
- "阿青的玉佩" → 与某个死者有关联
- "顾先生的真实身份" → 惊蛰旧人

### 意象伏笔
贯穿全书的象征/母题，不需要精确回收，但需要首尾呼应。

示例：
- 枯井 = 被掩盖的真相
- 药香 = 日常与安宁
- 匕首 = 杀手身份

### 情绪伏笔
暗示人物心理状态的细节，不需要明确回收，但读者回看时会恍然大悟。

示例：
- "沈渡的手顿了一下" → 暗示他内心震动
- "阿青的语气很平淡" → 暗示他见过更多世面

## 新伏笔埋设流程

1. 确定伏笔类型（剧情/人物/意象/情绪）
2. 确定埋设方式（隐蔽/适度/明显）
3. 确定计划回收章节
4. 在本表新增条目
5. 在章纲的"伏笔呼应层"中记录
