# 节点纲指南

节点纲是对**关键转折点**的超详细规划，比普通章纲详细3-5倍。

---

## 什么是节点

以下场景需要写节点纲：

| 节点类型 | 说明 | 示例 |
|---------|------|------|
| 主要转折点 | 改变故事走向的事件 | 主角身份暴露、盟友背叛 |
| 高潮场景 | 全书/全卷最有张力的段落 | 最终对决、真相揭晓 |
| 核心悬念揭晓 | S级或A级悬念的答案 | "凶手是谁""身世之谜" |
| 主角突破/觉醒 | 力量体系的关键升级 | 境界突破、获得新能力 |
| 反派对决 | 与主要反派的正面冲突 | 武力对决、智力博弈 |
| 情感爆发 | 人物关系的关键转折 | 师徒和解、恋人告白 |

---

## 节点纲的字段

### 基础信息
- 节点名称
- 所在章节
- 节点类型（转折/高潮/揭秘/突破/对决/情感）
- 前置铺垫（哪些章节为此节点做了铺垫）

### 情节设计
- 核心事件的详细描述
- 涉及的所有角色
- 每个角色的立场和行动
- 信息的揭露顺序（先让读者知道什么，后知道什么）

### 情绪节拍（每500字的情绪变化）

```
字数区间    情绪状态        具体描写方向
0-500字     紧张/期待       环境铺垫，气氛渲染
500-1000字  升温            冲突开始，各方就位
1000-1500字 爆发            核心冲突展开
1500-2000字 高潮            最强烈的瞬间
2000-2500字 余波            结果呈现，情绪回落
2500-3000字 钩子            留下新的悬念或回味
```

### 对话设计
- 关键对话的要点（不必写完整对话，但要确定核心台词）
- 金句设计（被读者传播的句子）
- 潜台词（角色说的和想的不一样）

### 动作/战斗设计
- 动作的节拍（试探→交锋→爆发→决胜）
- 环境破坏程度（体现力量等级）
- 旁观者反应（通过他人震惊烘托主角）

### 名场面设计
- 最有画面感的瞬间是什么？
- 这个瞬间如何被读者记住？
- 是否有"封神"潜力（如《遮天》的"叶天帝"封号场景）？

### 伏笔处理
- 本节点回收了哪些伏笔？
- 本节点埋下了哪些新伏笔？
- 是否有"认知颠覆"型反转？

## 节点纲示例

```
节点名称：沈渡身份暴露
所在章节：第26章
节点类型：转折+高潮

前置铺垫：
- 第1章：沈渡认出摘星手（暗示身份）
- 第8章：官府开始怀疑沈渡
- 第15章：沈渡对阿青坦白部分过往
- 第22章：阿青身世揭露，信任崩塌

情绪节拍：
0-500字：压抑——铁佛率众围住药铺，沈渡平静应对
500-1000字：升温——铁佛当众揭露沈渡的杀手身份
1000-1500字：爆发——镇民震惊，阿青难以置信
1500-2000字：高潮——沈渡承认，但说出真相的另一面
2000-2500字：余波——阿青的反应，镇民的态度变化
2500-3000字：钩子——阿青转身离开，沈渡没有追

关键台词：
- 铁佛："你们的陈三，十五年前叫沈渡，江湖人称渡鸦。"
- 沈渡："你说的都对。但你漏了一件事。"
- 阿青："……你为什么不告诉我？"

名场面设计：
沈渡站在药铺门口，背后是"陈记药铺"的匾额。铁佛的人围了三圈。
他没有拔刀，只是站在那里，像一把收进鞘里的刀。
"我杀过人。"他说，声音很平，像在说别人的事。
"但我没杀那个人。"
```

---

## 写节点纲的检查清单

- [ ] 前置铺垫是否充足？（至少3-5章的铺垫）
- [ ] 情绪节拍是否合理？（有起伏，不是一条直线）
- [ ] 是否有金句/名场面？
- [ ] 伏笔回收是否到位？
- [ ] 结尾是否有钩子？
- [ ] 人物行为是否符合设定？
- [ ] 是否有足够的"爽感"或"情感冲击"？
