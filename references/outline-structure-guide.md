# 大纲 / 卷纲 / 章纲 写作指南

本文档指导如何将故事概念逐层细化为可执行的章节规划。

---

## 一、总纲（Story Outline）

总纲是全书的宏观骨架，回答"这个故事讲什么、怎么讲"。

### 结构选择

根据故事类型选择合适的叙事结构：

**三幕式**
- 第一幕（25%）：建置——介绍人物、世界、核心冲突
- 第二幕（50%）：对抗——冲突升级、人物成长、反转
- 第三幕（25%）：解决——高潮、结局

**起承转合**
- 起（20%）：开端
- 承（30%）：发展
- 转（30%）：转折与高潮
- 合（20%）：收束

**英雄之旅**（适合成长型故事）
- 日常世界 → 冒险召唤 → 拒绝召唤 → 遇见导师 → 跨越第一道门槛
- 试炼、盟友与敌人 → 接近最深的洞穴 → 严峻的考验 → 获得奖赏
- 返回之路 → 复活 → 携万能药归来

**多线并行**（适合群像/史诗）
- 多条独立故事线同时推进
- 在关键节点交汇
- 最终汇聚或形成对比

### 主线冲突链

用以下模板梳理主线：

```
核心冲突：[一句话概括]
  ↓
冲突建立：[读者第一次意识到冲突的时刻]
  ↓
冲突升级1：[第一次加剧]
  ↓
冲突升级2：[第二次加剧，范围/强度/代价都更大]
  ↓
至暗时刻：[主角看似彻底失败]
  ↓
转折点：[主角找到突破的关键]
  ↓
高潮：[最终对决/解决]
  ↓
结局：[冲突解决后的新常态]
```

### 副线设计

每条副线需要明确：
- **主题**：这条副线在探讨什么
- **承载人物**：哪些角色主要负责这条线
- **与主线的交汇点**：在哪个阶段与主线碰撞
- **独立解决时机**：在主线高潮前/后/同时解决
- **对主线的影响**：副线如何推动或阻碍主线

### 全书情绪曲线

用文字描述全书的情绪走向，例如：

```
开篇（平静中暗藏危机）
  → 前期（好奇与期待，偶有紧张）
    → 前中期（第一次震撼，基调转暗）
      → 中期（压抑中缓慢积蓄力量）
        → 中后期（连环打击，至暗时刻）
          → 后期（曙光初现，热血攀升）
            → 高潮（全面爆发，极致张力）
              → 结尾（余韵悠长，归于平静）
```

## 二、卷纲（Volume Outline）

卷纲将总纲拆分为若干卷，每卷是一个相对独立的故事弧。

### 每卷必填项

| 字段 | 说明 |
|------|------|
| 卷名 | 本卷的名称或代号 |
| 主题词 | 本卷的核心主题（如"觉醒""背叛""崛起"） |
| 卷级目标 | 本卷结束时故事应推进到哪里 |
| 关键事件 | 3-5 个本卷必须发生的大事件 |
| 新出场人物 | 本卷新登场的重要角色 |
| 退场人物 | 本卷退场的角色（死亡/隐退/离开） |
| 推进的副线 | 本卷推进了哪些副线及推进内容 |
| 情绪曲线 | 本卷的情绪走向（与全书曲线的关系） |
| 卷末钩子 | 最后一章的悬念，驱动读者继续 |
| 预估字数 | 本卷的字数目标 |

### 卷与卷之间的衔接

- 前一卷的卷末钩子应在后一卷的前几章有所回应
- 时间跳跃需明确交代
- 新出场人物需有合理的引入方式
- 前一卷的遗留问题需在后一卷有所推进

### 常见的分卷策略

- **按成长阶段分**：每个修行境界/人生阶段一卷
- **按地理区域分**：每个重要地点一卷
- **按势力冲突分**：每个主要敌人/势力一卷
- **按时间跨度分**：每个重要时期一卷

## 三、章纲（Chapter Outline）

章纲是最细粒度的规划，每章都要填写完整的章纲卡片。

### 章纲字段详解

#### 基础层

| 字段 | 说明 | 示例 |
|------|------|------|
| chapter | 章节编号 | 1 |
| title | 章节标题 | "风起青萍" |
| volume | 所属卷 | 1 |
| pov | 视角人物 | "林渊" |
| location | 场景地点 | "青云城·林家废园" |
| timeline | 故事内时间 | "天元历347年·春·清晨" |

#### 人物层

| 字段 | 说明 | 示例 |
|------|------|------|
| characters_present | 出场人物 | ["林渊", "林父", "管家老周"] |
| character_focus | 重点刻画人物 | ["林渊"] |
| relationship_shift | 本章关系变化 | "林渊与父亲关系从冷漠→开始理解" |

#### 情节层

| 字段 | 说明 | 示例 |
|------|------|------|
| key_events | 关键事件 | ["林渊发现父亲的秘密信件", "家族会议宣布除名"] |
| conflict | 核心冲突 | "外部冲突：家族排斥；内部冲突：自我认同危机" |
| conflict_resolution | 冲突解决方式 | "部分解决——林渊决定离开但未完全释怀" |
| subplot_threads | 推进的副线 | ["身世之谜：发现信件暗示非亲生"] |
| chapter_hook | 章末钩子 | "林渊翻到信件最后一页，看到一个熟悉又陌生的名字" |

#### 结构层

| 字段 | 说明 | 示例 |
|------|------|------|
| opening_type | 开篇方式 | "描写开场——废园的荒凉景象" |
| closing_type | 收束方式 | "悬念收束——信件中的名字" |
| scene_breakdown | 场景分解 | ["1.废园独坐(静)", "2.家族会议(冲突)", "3.夜间翻信(转折)"] |
| pacing | 节奏 | "先慢后快——开头沉静，会议紧张，结尾急促" |
| narrative_technique | 叙事手法 | "顺叙为主，会议中穿插回忆" |

#### 情绪与氛围层

| 字段 | 说明 | 示例 |
|------|------|------|
| emotional_arc | 情绪弧 | "孤独→屈辱→震惊→决意" |
| tone | 基调 | "压抑→爆发→沉静" |
| sensory_focus | 感官侧重 | "视觉（废园的破败）+ 听觉（会议中的嘲讽声）" |
| atmosphere_keywords | 氛围关键词 | ["荒凉", "压抑", "暗流涌动", "月光清冷"] |

#### 伏笔与呼应层

| 字段 | 说明 | 示例 |
|------|------|------|
| foreshadowing | 埋下的伏笔 | ["信件中的名字（预计第15章回收）", "废园中的异响（预计第3章回收）"] |
| callback | 回收的伏笔 | ["序章中提到的'被遗忘的血脉'（第0章埋设）"] |
| symbol_motif | 象征/意象 | ["废园=林渊的处境", "月光=真相的微光"] |

#### 风格层

| 字段 | 说明 | 示例 |
|------|------|------|
| style_notes | 风格侧重 | "本章以环境描写和心理活动为主，对话精简有力" |
| dialogue_ratio | 对话占比 | "低（约20%）" |
| description_density | 描写密度 | "高" |
| word_count_target | 目标字数 | 4000 |

### 章纲检查清单

每章章纲完成后检查：
- [ ] 本章是否有明确的叙事目的（删掉它故事会少什么）？
- [ ] 章末钩子是否足够有力？
- [ ] 情绪弧是否与全书/全卷情绪曲线吻合？
- [ ] 伏笔的埋设和回收是否有计划？
- [ ] 场景分解是否合理（不会太碎或太长）？
- [ ] 节奏是否与前后章形成变化（避免连续多章同一节奏）？
- [ ] 视角人物是否合适（谁的视角最能展现本章冲突）？

## 四、层与层之间的关系

```
总纲（骨架）
  └→ 决定分几卷、每卷的宏观任务
      └→ 卷纲（血肉）
          └→ 决定每卷有几章、每章的事件分配
              └→ 章纲（皮肤）
                  └→ 决定每章的具体执行细节
                      └→ 创作（呼吸）
```

任何一层的修改都可能影响其下层。如果在写章纲时发现总纲有问题，应先回到总纲修正，再重新生成受影响的卷纲和章纲。

如果在创作过程中发现章纲需要调整，评估影响范围：
- 只影响本章 → 直接修改章纲
- 影响后续几章 → 修改后续章纲
- 影响全卷 → 回到卷纲层面修正
- 影响全书 → 回到总纲层面修正
