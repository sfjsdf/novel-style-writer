# 人物卡片模板

本文档定义角色设计的标准格式。分为「完整卡片」（主角/反派/核心配角）和「简化卡片」（次要角色）两种。

---

## 完整人物卡片（主角/反派/核心配角）

### 基础信息

| 字段 | 说明 |
|------|------|
| 姓名 | 全名，含字号/别名/代号 |
| 年龄 | 故事开始时的年龄 |
| 外貌 | 身高、体型、显著特征（2-3 句，突出辨识度） |
| 身份 | 社会角色、职业、地位 |
| 首次出场 | 第几卷第几章 |

### 性格内核

- **核心特质**（3-5 个关键词）：如"隐忍、聪慧、重情义、偏执、表里不一"
- **性格成因**：什么经历塑造了 ta 的性格（童年/创伤/教育/关键事件）
- **外在表现**：日常中如何展现这些特质（行为模式、社交风格）
- **内在矛盾**：性格中自相矛盾的地方（如"渴望自由却害怕孤独"）

### 动机与目标

- **表面目标**：ta 自己说得出的目标（如"变强""复仇""找到真相"）
- **深层需求**：ta 自己未必意识到的真正需求（如"被认可""安全感""赎罪"）
- **动机强度**：目标对 ta 有多重要（愿意付出什么代价）

### 缺陷与弱点

- **性格缺陷**：导致 ta 犯错的性格弱点（如"过于信任他人""优柔寡断"）
- **能力短板**：ta 不擅长什么
- **致命弱点**：如果被敌人利用，最致命的弱点是什么
- **恐惧**：最深层的恐惧是什么

### 人物弧光

分阶段描述角色的成长/变化轨迹：

| 阶段 | 状态 | 关键转变事件 |
|------|------|-------------|
| 故事开始 | ta 是什么样的人 | — |
| 第一阶段结束 | ta 发生了什么变化 | 什么事件导致了变化 |
| 中期 | ta 的核心冲突如何激化 | 关键考验 |
| 高潮前 | ta 面临什么抉择 | 最终选择 |
| 故事结束 | ta 成为了什么样的人 | 最终蜕变 |

### 标志性特征

- **口头禅**：常说的话（反映性格）
- **小动作**：习惯性动作（如紧张时转戒指、思考时咬指甲）
- **独特癖好**：让人物更立体的小特点
- **着装风格**：外表如何反映内在

### 秘密

- 人物隐藏的真相（至少一个）
- 这个秘密如何影响 ta 的行为
- 秘密在故事中何时/如何揭露

### 与主题的关系

- 这个角色如何体现小说的核心主题
- ta 的故事线如何服务于主题表达

## 简化人物卡片（次要角色）

| 字段 | 说明 |
|------|------|
| 姓名 | 姓名 + 称呼 |
| 身份 | 职业/地位/与主角的关系 |
| 核心性格 | 1-2 句话概括 |
| 故事功能 | 导师/对手/催化剂/信息源/保护者/阻碍者/…… |
| 标志特征 | 一个让人记住 ta 的特征（外貌/语言/行为均可） |
| 出场卷数 | 第几卷出场 |
| 退场方式 | 死亡/隐退/背叛/远行/融入日常/…… |
| 与主角的关系变化 | 初期关系 → 最终关系 |

---

## 反派补充要素

在完整卡片基础上，反派还需补充：

- **反派逻辑**：ta 为什么认为自己是对的（反派的自我辩护）
- **镜像关系**：ta 与主角在什么维度上是对照（如同样经历不同选择）
- **魅力点**：这个反派有什么吸引人的地方（能力/魅力/理念/悲剧性）
- **底线**：ta 有没有不做之事（即使反派也应有底线，否则扁平）
- **结局**：死亡/改过/逍遥法外/被感化/同归于尽/……

---

## 人物设计检查清单

- [ ] 人物是否有至少一个让人记住的特征？
- [ ] 人物动机是否清晰且可理解？
- [ ] 人物是否有内在矛盾（非完美角色）？
- [ ] 人物弧光是否有足够的铺垫支撑？
- [ ] 人物在关系网中是否有明确定位？
- [ ] 人物数量是否合理（是否有可合并的角色）？
- [ ] 反派是否足够有深度（不只是"坏"）？
- [ ] 女性角色/配角是否有独立人格（非工具人）？
